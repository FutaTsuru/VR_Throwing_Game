# システム概要 
VR_gameは、VR上で敵の攻撃物をHMD（頭）を使って避けながら、
手に持っているボールをコントローラーを振りかざすことで投げ、的を射るゲームです。
攻撃物を避けることでダメージを避け、時々投げるハートに当たると回復する。
的には点数が書かれており、状況に応じて投げる方向や強さを判断する必要がある。
